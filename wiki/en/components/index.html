What are Components?

Components are one from the main game features. Each component is doing something, for example "Network Component" ( we will talk about component types later) increases your upload speed, "Storage" one increases size of the uploads.

Component Types

There are 4 main component types, unlocked by default:

Network Component
Storage Component
Mining Cursor Component
Processor Component
Increases upload speed what is needed to complete Uploads. Increases size of the Uploads, means it will give more reputation after completing and so on. Doubles credits from Mining feature. Increases maximum amount of Uploads that can be uploaded at the same time.

Other Component Types

However there are 2 other component types that are NOT unlocked by default. You can unlock them manually in Laboratory via Research Points.

Manufactory Component
Overclocking Component
