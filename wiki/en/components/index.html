What are Components?

Components are one of the game's main features. Each component does something, for example "Network Component" (we will talk about component types later) increases your upload speed, and "Storage components" increases size of the uploads.

Component Types

There are 4 main component types, unlocked by default:

Network Component
Storage Component
Mining Cursor Component
Processor Component
Increases upload speed. It is needed to complete uploads. Increases size of the uploads, which increases reputation gain once uploading is completed. Doubles credits from mining Increases the maximum amount of uploads that can be completed simultaneously.

Other Component Types

There are 2 other component types that are NOT unlocked by default. You can unlock them manually in the Laboratory using research points.

Manufactory Component
Overclocking Component
